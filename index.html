Portail des projets IUT

Ce portail centralise les dépôts GitHub et les pages Hackster.io pour chaque SAÉ de l'IUT.


Un pipeline CI/CD est configuré via GitHub Actions pour automatiser les tests unitaires et le déploiement Docker.


La structure front-end utilise React et Redux, interrogeant une API RESTful sécurisée par JWT.


Les métadonnées sont indexées dans ElasticSearch, offrant une recherche full-text et des filtres par tags.


Un monitoring Grafana se connecte à Prometheus pour suivre la santé des conteneurs et générer des alertes.

Potify

Développée en Java dans Android Studio, Potify adopte l’architecture MVVM pour modulariser le code.


La base Room assure une gestion robuste des données locales, tandis que Coroutine gère les appels asynchrones.


WorkManager orchestre les tâches de téléchargement et de mise en cache des pistes audio.


L’UI repose sur RecyclerView et Material Design Components pour une expérience utilisateur fluide.

Jeu du Pendu

Cette application web utilise HTML5 Canvas pour dessiner le pendu et gérer les animations.


Le backend en Node.js/Express déploie une API sécurisée par JSON Web Tokens pour l’authentification.


Les données utilisateurs et scores sont stockées dans MongoDB Atlas, avec Mongoose pour schéma et validation.


Le front-end Vanilla JS utilise fetch pour communiquer en AJAX et mettre à jour l’interface en temps réel.

Pipeline de données

Orchestration Apache Airflow avec DAGs planifiés pour ingérer des données depuis divers endpoints REST.


Les tâches Python exploitent Pandas et NumPy pour transformer et nettoyer les flux bruts.


Les résultats sont poussés dans PostgreSQL, optimisés par partitionnement et index GIN pour requêtes analytiques.


La supervision des exécutions se fait via Grafana, remontant les métriques Prometheus pour alertes SLA.
