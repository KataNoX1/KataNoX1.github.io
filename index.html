Evitech


Evitech est une entreprise spécialisée dans les solutions de sécurité vidéo intelligentes.

J’ai effectué un stage d’un mois au sein d’Evitech, débutant par une formation d’une semaine sur l’outil Jaguar, leur logiciel d’analyse vidéo.

Ensuite, j’ai développé en Python un script utilisant la bibliothèque os pour surveiller l’arrivée complète de vidéos, les déplacer puis lancer Jaguar.

La documentation en ligne manquait, j’ai donc contacté les développeurs pour résoudre les blocages techniques.

J’ai terminé le projet en modifiant une page de leur intranet (reStructuredText et Sphinx) pour y ajouter un système de tags, améliorant la recherche des vidéos d’exemple.

CPCP Télécom
